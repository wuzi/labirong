# Labirong

> A multiplayer labyrinth racing game, feel free to open a pull request to add snippets, optimizations, add features, etc.

#### Build Setup

``` bash
# install dependencies
npm install

# serve at localhost:8080
npm start

```

# Live demo

Check it [here](http://labirong.herokuapp.com/)!
